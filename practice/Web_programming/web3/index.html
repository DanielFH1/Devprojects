About Semantic HTML

Tags Description
header the top of the page, usually the topic places
nav navigation to specific page of website such as 'about us' , login, signup etc...
footer the bottom of the page, usually contact info, name etc...
main article: just like "section", but it can stand independently
section: just like "article", but it has to be related
figure&figcaption: you can insert an image and put some description by figcaption
anchor: you can establish an hyperlink in it.
paragraph : separated from adjacent blocks.
